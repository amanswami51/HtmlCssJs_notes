24. block tags and inline tags

A block level element always starts on a new line and takes up the full width available.

block tags are :- (div, address, article, aside, blockquote, canvas>, dd, dl, dt, fieldset, figcaption, figure, footer, form, h1, h6, header, hr, li, main, nav, noscript, ol, p, pre, section, table, tfoot, ul, video)

An inline element does not start on a new line and it only takes up as much width as necessary.

inline tags are :- (span, a, abbr, acronym, b, bdo, big, br, button, cite, code, dfn, em, i, input, kbd, label, map, object, output, q, samp, script, select, small, strong, sub, sup, textarea, time, tt, var)
